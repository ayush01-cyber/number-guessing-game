# Number Guessing Game

A simple command-line Number Guessing Game built using Python.

The program generates a random number between 1 and 100, and the player must guess the correct number. The game provides hints if the guess is too high or too low and keeps track of the number of attempts taken to win.

---

## Features

- Random number generation
- User input handling
- High and Low hints
- Attempt counter
- Input validation using try-except
- Exception handling with ValueError
- Simple command-line interface
